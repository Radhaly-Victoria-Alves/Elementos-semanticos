O que é a tag details?

A tag <details> é usada para criar um widget de interface do usuário que pode ser expandido ou recolhido. Ela é frequentemente usada para mostrar ou esconder informações adicionais, como descrições, instruções ou detalhes extras sobre um tópico específico.

Como usar a tag details?

Para usar a tag <details>, você deve incluir um elemento <summary> que serve como o título ou cabeçalho do conteúdo que será mostrado ou escondido. O conteúdo dentro da tag <details> será exibido quando o usuário clicar no resumo.
